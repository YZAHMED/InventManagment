## Wireframe Notes:

### Design Principles:
- **Simple and Clean**: Easy to understand and navigate
- **Consistent Layout**: Same header and navigation across all pages
- **Clear Actions**: Buttons are clearly labeled and positioned
- **Responsive**: Works on different screen sizes
- **User-Friendly**: Intuitive flow from one page to another

### Color Coding:
- **Green Badges**: High quantity (10+ items)
- **Yellow Badges**: Medium quantity (1-9 items)  
- **Red Badges**: Low quantity (0 items)

### Navigation Flow:
1. Home → Login/Register
2. Login → Inventory List
3. Register → Login → Inventory List
4. Inventory List → Add/Edit/Delete pages
5. All pages → Back to Inventory List

### Key Features Shown:
- User authentication status in header
- Image upload functionality
- Quantity tracking with visual indicators
- Confirmation dialogs for destructive actions
- Empty state handling
